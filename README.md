CAMPUS AFFILIATE MARKETPLACE

📌 Project Overview

Campus Affiliate Marketplace is an AI-driven institutional affiliate revenue platform developed using Microsoft Power Platform technologies.

The system allows colleges to manage affiliate products and enables students to browse, search, wishlist, and purchase products through a centralized campus application.

When students purchase products through affiliate links, the institution earns commission from platforms like Amazon, Flipkart, Myntra, and other partner platforms.


🚀 Technologies Used

- Microsoft Power Apps
- Microsoft SharePoint
- Microsoft Power Automate
- Microsoft Power BI
- Microsoft Copilot Studio
- Affiliate Platforms (Amazon / Flipkart / Myntra)

🎯 Problem Statement

Currently, students purchase products individually from e-commerce platforms without any centralized institutional system.

Colleges are unable to utilize affiliate marketing opportunities for:
- revenue generation
- click tracking
- analytics
- centralized product management

This project solves the problem by creating a centralized affiliate marketplace platform.

💡 Proposed Solution

The Campus Affiliate Marketplace provides:

- Product browsing system
- Affiliate product management
- Wishlist functionality
- Product sharing feature
- Click tracking automation
- Analytics dashboard
- AI chatbot assistance

The system automatically tracks clicks and redirects users through affiliate links, allowing institutions to earn commission.

🏗️ System Architecture & Workflow

Workflow Steps

1. Admin logs into Power Apps
2. Admin creates affiliate products
3. Product data stored in SharePoint
4. Students browse/search products
5. Students add products to wishlist
6. User clicks Buy button
7. User redirected to affiliate platform
8. Purchase completed
9. Power Automate tracks clicks
10. Data stored in SharePoint
11. College earns commission
